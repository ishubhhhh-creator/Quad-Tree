# Quad-Tree
This project focuses on the design and implementation of a Quadtree, a hierarchical data structure specifically engineered for efficient two-dimensional spatial partitioning. The core logic involves the recursive decomposition of a 2D coordinate space into four distinct quadrants—North-West, North-East, South-West, and South-East—whenever a specific node exceeds its predefined point capacity. By utilizing this tree-based approach, the implementation significantly optimizes spatial queries, such as range searches and nearest-neighbor lookups, reducing the time complexity from a naive $O(n)$ linear scan to an average of $O(\log n)$. The project emphasizes rigorous memory management in C++, ensuring that pointers are correctly handled during subdivision and that all dynamically allocated nodes are properly deallocated to prevent leaks. This structure serves as a foundational component for applications in geographic information systems (GIS), collision detection in game engines, and efficient image compression.
